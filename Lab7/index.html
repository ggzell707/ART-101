Challenge

For this lab, we had to create a function. We created one based on the example on canvas that retreives user input and outputs a line of text followed by the user input sorted alphabetically.

Problems

My partner and I had a hard time reaching our desired output, mostly due to spelling errors. The tasks made me realize that when practicing JavaScript, it is very important to go back and check that everything is written properly. It's crucial to pay attention to small details like capitalization or punctuation when writing code, as even minor errors can cause the code to fail. To solve this issue, we would check each other's work afterward to look for errors when our code would not run. We also offered each other feedback and advice, such as organizing our files properly or suggesting memorization tips.

Reflection

This activity made me more aware of having to type properly to successfully run my program. It also made me realize that the more I dive into the coding world, the more I will have to debug.
