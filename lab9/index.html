Lab 9 - Libraries and Jquery

Challenge

For this lab, we had to create three buttons for our sections by connecting the Javascript to CSS and HTML using Jquery.

Problems

We came across a couple of problems when completing this assignment. The instructions were a bit unclear about what elements go into which folder, so we had to figure that out through trial and error.

Results

We ended up creating three buttons, each one at the bottom of every section. Upon clicking, the minor section box changes in color. This was possible through using Jquery.
