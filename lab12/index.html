Lab 12: Conditionals

Challenge

The challenge of this project was to use conditionals to create a input text box that outputs a specific output based on the characters of the input.

Problems

One problem we faced was that the output would not show up upon request. We realized that it was because the HTML and Javascript files were not properly linked to each other. To solve this issue, we used chatGPT to write the label for the input so that way it would show up when we opened the website.

.

Results

The results are down below. It is a textbox in which you can write your name, and click submit, and it will print out the beginning of a sentence and a different result at the end based on the characters of the input.
